    Aulas síncronas 2024

    Faremos 15 encontros para as aulas síncronas em formato de live no meu canal do YouTube entre as datas de 11/06 e 30/07.

    Como vai funcionar?

    Nossos encontros acontecerão as terças e quintas com duração de 1h30m. Entre às 20:00h e 21:30. Com chat aberto para tirar dúvidas enquanto a aula acontece. O material base é o que está disposto neste site.

    Caso queira conversar e tirar dúvidas com as pessoas que também estão/irão fazer o curso, temos um grupo no Telegram.

    O que será necessário para acompanhar?

    1. Um link com internet para participar das lives
    2. Um computador para acompanhar as aulas
    3. Um sistema operacional que seja capaz de rodar o Docker (algumas versões do Windows tem problemas com isso)
    4. Uma conta no GitHub para hospedar os códigos e exercícios
    5. Uma conta no Fly.io para fazer o deploy de sua aplicação
